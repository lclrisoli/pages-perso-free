Liste des transferts

Codage d'un fichier sous Windows
Encodage: certutil -encode file.png file.png.txt
Décodage: certutil -decode file.png.txt file.png

Codage d'un fichier sous Linux
Encodage: uuencode -m file.png file.png > file.png.txt
Décodage: uudecode -o file.png file.png.txt

Manipulation de fichier sous Linux
Fractionner: split -l 10000 file.png.txt file.png.txt_

Fusion:
dos2unix file.png.txt_*
cat file.png.txt_* > file.png.txt
